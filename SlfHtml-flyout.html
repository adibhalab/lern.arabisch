Flyout-Menü mit toggle-Button

Um Platz für Inhalte zu schaffen, soll die Navigation nur bei Bedarf eingeblendet werden. Deshalb erweitern wir unsere Flyout-Navigation um einen toggle-Button.

  1. Im Grundzustand ohne CSS ist alles sichtbar - der Button ist ohne Funktion.
  2. Mit CSS wird die Seite gestaltet und der Button ausgeblendet.
    Die Navigation ist bedienbar, aber immer sichtbar!
  3. In dieser Version ist nur ein Navicon sichtbar - erst ein Klick auf dieses öffnet das Menü.
